🛒 Retail Sales Dashboard
📌 Project Overview

This project is an interactive Retail Sales Dashboard designed to analyze sales performance, inventory, revenue trends, and promotional impact across different regions and channels.

The dashboard helps businesses monitor KPIs and make data-driven decisions efficiently.

🚀 Features
Interactive slicers and filters
Revenue analysis by month
Units sold tracking
Delivery quantity monitoring
Stock availability analysis
Average unit price calculation
Region-wise revenue contribution
Channel-wise revenue share
Promotional vs non-promotional sales comparison
📊 KPIs Included
KPI	Description
Total Revenue	Overall generated revenue
Unit Sold	Total units sold
Delivery Quantity	Total delivered products
Stock Available	Current inventory availability
Average Price Unit	Average selling price
🛠 Tools & Technologies
Microsoft Excel / Microsoft Power BI
Pivot Tables
Charts & Graphs
Slicers
Data Cleaning
KPI Dashboarding
📈 Dashboard Insights
Identified top-performing regions
Compared sales across channels
Evaluated promotional campaign effectiveness
Analyzed monthly revenue trends
Monitored stock availability
🎯 Business Use Case

This dashboard can help retail businesses:

Improve inventory planning
Track sales growth
Optimize promotional campaigns
Understand customer buying trends
Support management reporting
